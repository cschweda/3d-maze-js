This editor allows you to create and modify maze files for the TRS-80 Style Maze game.

Getting Started

  • Enter maze dimensions and click "Create New Maze" to generate a random maze
  • Use the editing tools to modify the maze structure
  • Set player start position and exit location
  • Click "Validate Maze" to ensure it's solvable
  • Use "Save Maze" to store your creation

Keyboard Shortcuts

  • W - Wall tool
  • P - Path tool
  • S - Start point tool
  • E - Exit point tool
  • +/- - Zoom in/out
  • Ctrl+S - Save maze
  • Ctrl+O - Open maze
